# Как разместить приложение на GitHub Pages

Чтобы ваше приложение стало доступно в интернете, выполните следующие шаги:

## 1. Подготовка на GitHub
1. Зайдите на сайт [GitHub.com](https://github.com) и войдите в свой аккаунт.
2. Создайте **новый репозиторий** (New Repository).
   - Назовите его, например, `metaphorical-cards`.
   - Сделайте его **Public** (пуличным), чтобы GitHub Pages был бесплатным.
   - Не добавляйте README или gitignore при создании (мы будем использовать наши).

## 2. Загрузка кода
В папке с вашим проектом на компьютере откройте терминал и выполните команды (замените `ВАШ_ЛОГИН` на ваш никнейм на GitHub):

```bash
# Инициализация git (если еще не сделано)
git init
git add .
git commit -m "Initial commit"

# Привязка к удаленному репозиторию
git branch -M main
git remote add origin https://github.com/ВАШ_ЛОГИН/metaphorical-cards.git
git push -u origin main
```

## 3. Настройка GitHub Pages
1. После того как код загрузится, перейдите в настройки репозитория на GitHub (**Settings**).
2. В боковом меню выберите **Pages**.
3. В разделе **Build and deployment** -> **Source** выберите **GitHub Actions**.
   - GitHub может автоматически определить наличие файла `.github/workflows/deploy.yml`, который я уже добавил в проект.
4. Процесс запустится автоматически. Перейдите во вкладку **Actions** вверху репозитория, чтобы следить за прогрессом.
5. Когда кружок станет зеленым, ваше приложение будет доступно по ссылке, которая появится в разделе **Settings -> Pages**.

Ссылка будет выглядеть примерно так:
`https://ВАШ_ЛОГИН.github.io/metaphorical-cards/`
